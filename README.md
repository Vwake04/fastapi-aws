# fastapi-aws

Github Actions + EC2 + Autoscaling Group + S3 + CodeDeploy 


## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Vwake04/fastapi-aws.git
    cd astapi-aws
    ```

2. Install dependencies:
    ```bash
    pip install -e .
    ```

## Running Tests

1. Install `pytest` if not already installed:
    ```bash
    pip install pytest
    ```

2. Run the tests:
    ```bash
    pytest
    ```
